Element - text and value

To get or set an element's text content, use the .textContent property

Syntax:

Element.textContent

Examples:

Get an element's text content. And save it to a variable.

var brainy = document.querySelector(".brainy");
var text = brainy.textContent;

Change an element's text content.

var clumsy = document.querySelector(".clumsy");
clumsy.textContent = "Whoops!";
